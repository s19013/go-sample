# フィクスチャーとは
テストを実行するために事前に用意するデータや状態のこと

# フィクスチャーのメリット
* テストが再現可能になる
* 読みやすくなる
* データの使い回しができる
* テストの意図が明確になる

# モックとの違い

| 種類      | 内容        |
| ------- | --------- |
| フィクスチャー | テストデータや状態 |
| モック     | 外部依存の偽物   |

* フィクスチャー =「材料」
* モック =「代用品」
